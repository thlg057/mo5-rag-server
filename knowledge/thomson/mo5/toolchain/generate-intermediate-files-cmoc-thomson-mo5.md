# Generate CMOC intermediate files for debugging

## Goal
Keep CMOC’s intermediate outputs (`.s`, `.lst`, `.map`, `.link`) to debug codegen and linking.

## Procedure
1. Enable intermediate generation:
   - `cmoc --intermediate --thommo foo.c`
   - Short form: `cmoc -i --thommo foo.c`
2. (Optional) Write intermediates to a dedicated directory (since CMOC 0.1.62):
   - `cmoc -i --intdir=build --thommo foo.c`

## What files you get
- When compiling (with `-i`):
  - `.s` (assembly generated by CMOC)
  - `.lst` (listing produced by `lwasm`)
- When linking (with `-i`):
  - `.map` (link map from `lwlink`)
  - `.link` (link script used by `lwlink`)
