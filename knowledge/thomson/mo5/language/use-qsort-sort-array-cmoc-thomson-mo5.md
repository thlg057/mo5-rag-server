# Use qsort() to sort an array (CMOC, Thomson MO5)

Sort a C array using quicksort.

## Steps

1. Include `<cmoc.h>`.
2. Provide a comparison function.
3. Call `qsort(base, n, size, cmp)`.

## Notes

- CMOC's `qsort()` uses recursion, so keep an eye on stack usage on MO5.
