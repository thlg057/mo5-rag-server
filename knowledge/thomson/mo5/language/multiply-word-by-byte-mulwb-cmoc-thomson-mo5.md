# Multiply a 16-bit word by an 8-bit byte with mulwb() (CMOC, Thomson MO5)

Multiply a 16-bit word by an 8-bit byte using CMOC's optimized helper.

## Steps

1. Include `<cmoc.h>`.
2. Call `mulwb(hi, w, b)`.

## Notes

- Refer to `cmoc.h` for the exact parameter meaning on your CMOC version.
