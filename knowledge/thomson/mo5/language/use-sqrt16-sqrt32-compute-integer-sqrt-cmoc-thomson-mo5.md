# Use sqrt16() and sqrt32() to compute integer square roots (CMOC, Thomson MO5)

Compute the integer square root of a 16-bit or 32-bit value using CMOC's helpers.

## Steps

1. Include `<cmoc.h>`.
2. Call `sqrt16(n)` for 16-bit inputs or `sqrt32(n)` for 32-bit inputs.

## Notes

- These functions return integer results.
